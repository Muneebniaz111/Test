## 🎯 Project Overview

AGAREO is a comprehensive real estate platform built with vanilla HTML, CSS, and JavaScript. It provides users with an elegant interface to discover properties, explore neighborhoods, and access information about real estate opportunities across the UAE. The application is fully responsive and optimized for all device sizes.

## ✨ Features

- **Responsive Design**: Fully mobile, tablet, and desktop responsive
- **Interactive Navigation**: Smooth scrolling and intuitive navigation bar
- **Property Showcase**: Beautiful display of property listings with detailed information
- **Services Section**: Comprehensive overview of available services
- **Testimonials**: Client reviews and testimonials section
- **Contact & CTA**: Clear call-to-action sections for user engagement
- **Modern UI/UX**: Clean and professional design following modern web standards
- **Performance Optimized**: Fast loading times and smooth interactions

## 🛠️ Tech Stack

- **HTML5**: Semantic markup and structure

- **Bootstrap 5**: Responsive grid system and components
- **Icons & Fonts**: Custom fonts and icon libraries

## 📦 Setup Instructions

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- A code editor (VS Code recommended)
- No build tools or dependencies required for basic setup

### Installation Steps

1. **Clone the Repository**
   ```bash
   git clone https://github.com/Muneebniaz111/Test.git
   cd Test
   ```

2. **Open the Project**
   - Navigate to the project folder
   - Open `index.html` directly in your web browser, or
   - Use VS Code's Live Server extension for development

3. **Using Live Server (Recommended)**
   - Install the "Live Server" extension in VS Code
   - Right-click on `index.html`
   - Select "Open with Live Server"
   - The application will open at `http://localhost:5500`

## 🚀 Running the Project

### Option 1: Direct Browser Access
```bash
# Simply open the index.html file in your browser
# No server setup required
```

### Option 2: Using Python (Local Server)
```bash
# Python 3.x
python -m http.server 8000

# Python 2.x
python -m SimpleHTTPServer 8000

# Then visit http://localhost:8000
```

### Option 3: Using Node.js (http-server)
```bash
# Install http-server globally
npm install -g http-server

# Run the server
http-server

# Visit http://localhost:8080
```
